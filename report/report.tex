\section{Task 1: Score Distillation Sampling (SDS)}

SDS generates an image from a text prompt by optimizing a latent that is
initialized as random Gaussian noise ($x^0 \sim \mathcal{N}(0, I)$, shape
$1\times4\times64\times64$). At each step we add noise at a random timestep
(DDIM Eq.\ 4), predict it with the frozen diffusion model under classifier-free
guidance, and update $x^0$ with the SDS gradient
$\nabla_{x^0} L = \mathbb{E}_{t,\epsilon}\!\left[w(t)\,(\epsilon_\theta(x^t, c, t) - \epsilon)\right]$,
with $w(t) = 1 - \bar{\alpha}_t$. We use the fixed assignment hyperparameters:
\texttt{guidance\_scale = 25}, 500 optimization steps, Stable Diffusion 2.1 base.

\paragraph{Evaluation.} CLIP ViT-B/32 similarity between each generated image and
its \textbf{prompt} (via \texttt{eval.py}).

\paragraph{Overall CLIP score.} \textbf{0.301}

\section{Task 2: Posterior Distillation Sampling (PDS)}

PDS edits source images toward a target prompt by matching stochastic latents
($\tilde{z}^t$) between source and target, with shared noise samples
$\epsilon^t$ and $\epsilon^{t-1}$. We use the fixed assignment hyperparameters:
\texttt{guidance\_scale = 7.5}, 500 optimization steps, Stable Diffusion 2.1 base.

\paragraph{Evaluation.} CLIP ViT-B/32 similarity between each edited image and its
\textbf{edit prompt} (via \texttt{eval.py}).

\paragraph{Overall CLIP score.} \textbf{0.326}
